# <a name="ml-fatal-error-a1008"></a>ML の致命的なエラー A1008

**一致しないマクロの入れ子**

マクロが、ファイルの末尾の前に終了しなかったか、または終了ディレクティブ[Endm](endm.md)がマクロブロックの外部で見つかりました。

このエラーの原因の1つは、の前にドットが省略されていることです[。](dot-repeat.md)または[を繰り返します。しばらくお待ち](dot-while.md)ください。
