# <a name="operator-mask"></a>MASK 演算子

*Recordfieldname*または*record*のビットが設定され、他のすべてのビットがクリアされているビットマスクを返します。

## <a name="syntax"></a>構文

> **マスク**{*recordfieldname* | *レコード*}
