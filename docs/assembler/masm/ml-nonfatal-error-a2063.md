# <a name="ml-nonfatal-error-a2063"></a>ML の致命的でないエラー A2063

**2の累乗のみに揃える: 式**

[ALIGN](align-masm.md)ディレクティブで指定された式が無効です。

**ALIGN**式は、2 ~ 256 の2の累乗でなければなりません。また、現在のセグメント、構造体、または共用体のアラインメントの値以下である必要があります。
