# <a name="real8"></a>REAL8

各*初期化子*について、倍精度 (8 バイト) の浮動小数点数を割り当てて、必要に応じて初期化します。

## <a name="syntax"></a>構文

> *名前* **REAL8** *initializer* ⟦ __、__ *初期化子*...⟧
