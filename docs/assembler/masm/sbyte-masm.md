# <a name="sbyte"></a>SBYTE

各*初期化子*のストレージの符号付きバイトを割り当てて、必要に応じて初期化します。 型が有効である任意の場所で、型指定子として使用することもできます。

## <a name="syntax"></a>構文

> *名前*  **SBYTE** *初期化子*⟦__、__ *初期化子*...⟧
