# <a name="for"></a>FOR

各*引数*に対して1回繰り返されるブロックをマークします。各繰り返しでは、現在の*引数*が*パラメーター*を置換します。

## <a name="syntax"></a>構文

> *パラメーター* ⟦ **: REQ** |  __: =__ *default*⟧ **、\<** _引数_⟦ __、__ *argument* ...⟧ **>** \
> *ステートメント*の\
> **ENDM**

## <a name="remarks"></a>コメント

[IRP](irp.md)と同じです。
