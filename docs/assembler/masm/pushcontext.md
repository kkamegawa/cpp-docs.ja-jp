# <a name="pushcontext"></a>PUSHCONTEXT

現在の*コンテキスト*の一部またはすべてを保存します。セグメントレジスタでは、基数値、リスティングおよび cref フラグ、またはプロセッサ/コプロセッサ値が想定されます。 *コンテキスト*は、(32 ビット masm のみ)、**基数**、**リスティング**、 **CPU** (32 ビット Masm のみ)、または**すべて**を**想定**できます。

## <a name="syntax"></a>構文

> **Pushcontext** *コンテキスト*
