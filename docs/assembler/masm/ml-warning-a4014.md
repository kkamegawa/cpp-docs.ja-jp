# <a name="ml-warning-a4014"></a>ML の警告 A4014

BSS セグメントでサポートされていない命令および初期化データ

BSS セクション内に初期化されたデータを定義しようとしました。  BSS セクションは、名前が BSS であるクラスとして定義されます。  これには、`.data?`の簡略化されたセグメントが含まれます。
