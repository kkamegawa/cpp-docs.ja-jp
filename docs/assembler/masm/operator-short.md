# <a name="operator-short"></a>SHORT 演算子

*ラベル*の種類を short に設定します。 *ラベル*へのすべてのジャンプは、短い形式である必要があります (-128 ~ + 127 バイトの範囲内で、ジャンプ命令から*ラベル*に)。

> **短い***ラベル*
