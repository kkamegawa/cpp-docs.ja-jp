# <a name="operator-"></a>;; 演算子

マクロ定義にのみ出現するマクロで、*テキスト*をコメントとして扱います。 この一覧には、マクロが展開されている*テキスト*は表示されません。

## <a name="syntax"></a>構文

> **;;** *テキスト*
