# <a name="operator-imagerel"></a>IMAGEREL 演算子

*式*の画像相対オフセットを返します。

## <a name="syntax"></a>構文

> **Imagerel** *式*

## <a name="remarks"></a>コメント

結果として得られる値は、多くの場合、RVA または相対仮想アドレスと呼ばれます。

IMAGEREL は、COFF オブジェクトの出力でのみ使用できます。
