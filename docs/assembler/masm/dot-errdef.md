# <a name="errdef"></a>.ERRDEF

*名前*が以前に定義されたラベル、変数、またはシンボルの場合、エラーを生成します。

## <a name="syntax"></a>構文

> **.Errdef** *name* ⟦ __,__ *message*⟧
