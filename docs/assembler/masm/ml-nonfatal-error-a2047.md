# <a name="ml-nonfatal-error-a2047"></a>ML の致命的でないエラー A2047

**空 (null) の文字列**

文字列は、引用符の区切り記号のペアであり、内に文字がありません。

文字列が有効であるためには、1-255 文字が含まれている必要があります。
