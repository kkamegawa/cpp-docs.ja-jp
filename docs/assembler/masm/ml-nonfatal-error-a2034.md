# <a name="ml-nonfatal-error-a2034"></a>ML の致命的でないエラー A2034

**セグメントブロック内にある必要があります**

次のいずれかがセグメントブロックの外部で見つかりました:

- 命令

- ラベル定義

- [この](operator-this.md)演算子

- [$](dollar.md)演算子

- プロシージャの定義

- [ALIGN](align-masm.md)ディレクティブ

- [ORG](org.md)ディレクティブ
