# <a name="dw"></a>DW

各*初期化子*について、1つの単語 (2 バイト) を割り当てて、必要に応じて初期化します。 型が有効である任意の場所で、型指定子として使用することもできます。 **DW**は[WORD](word.md)のシノニムです。

## <a name="syntax"></a>構文

> ⟦*name*⟧ **DW** *initializer* ⟦ __、__ *initializer* ...⟧
