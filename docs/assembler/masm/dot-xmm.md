# <a name="xmm-32-bit-masm"></a>.XMM (32 ビット MASM)

インターネットストリーミング SIMD 拡張命令のアセンブリを有効にします。 (32 ビット MASM のみ。)

## <a name="syntax"></a>構文

> **.XMM**
