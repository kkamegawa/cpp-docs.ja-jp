# <a name="endm"></a>ENDM

マクロまたは繰り返しブロックを終了します。

## <a name="syntax"></a>構文

> **ENDM**

## <a name="remarks"></a>コメント

「[マクロ](macro.md)、 [for](for-masm.md)、 [forc](forc.md)、 [REPEAT](repeat.md)、または[WHILE](while-masm.md)」を参照してください。
