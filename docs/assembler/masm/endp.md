# <a name="endp"></a>ENDP

プロシージャ名の最後に、 **PROC**で開始されたプロシージャ*名*をマークします。

## <a name="syntax"></a>構文

> *名前* **endp**

## <a name="remarks"></a>コメント

「 [PROC](proc.md)」を参照してください。
