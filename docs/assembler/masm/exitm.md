# <a name="exitm"></a>EXITM

現在の繰り返しまたはマクロブロックの展開を終了し、ブロックの外側にある次のステートメントのアセンブリを開始します。

## <a name="syntax"></a>構文

> **Exitm** ⟦*textitem*⟧

## <a name="remarks"></a>コメント

マクロ関数では、 *textitem*は返される値です。
