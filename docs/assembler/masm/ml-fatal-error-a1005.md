# <a name="ml-fatal-error-a1005"></a>ML の致命的なエラー A1005

**アセンブラーの制限: マクロパラメーター名テーブルがいっぱいです**

マクロに対して定義されたパラメーター、ローカル、またはマクロのラベルが多すぎます。 マクロ名テーブルに空き領域がありませんでした。

短い名前または少ない名前を定義するか、不要なマクロを削除します。
