# <a name="dword"></a>DWORD

各*初期化子*に対して、2つの単語 (4 バイト) を割り当てて、必要に応じて初期化します。 **DWORD**は、 [DD](dd.md)のシノニムです。

## <a name="syntax"></a>構文

> ⟦*name*⟧ **DWORD** *初期化子*⟦ __,__ *initializer* ...⟧

## <a name="remarks"></a>コメント

型が有効である任意の場所で、型指定子として使用することもできます。
