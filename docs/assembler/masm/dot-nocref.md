# <a name="nocref"></a>.NOCREF

シンボルテーブルとブラウザーファイル内の記号の一覧を表示しません。

## <a name="syntax"></a>構文

> **.Nocref** ⟦*name*⟦ __、__ *name* ...⟧⟧

## <a name="remarks"></a>コメント

名前を指定した場合は、指定した名前だけが抑制されます。 と同じ[です。XCREF](dot-xcref.md)。
