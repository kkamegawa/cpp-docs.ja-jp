# <a name="tbyte"></a>TBYTE

*初期化子*ごとに10バイトのストレージを割り当て、必要に応じて初期化します。 型が有効である任意の場所で、型指定子として使用することもできます。 **DT**は、 **t**のシノニムです。

## <a name="syntax"></a>構文

> ⟦*name*⟧ **t** *初期化子*⟦ __、__ *初期化子*...⟧
