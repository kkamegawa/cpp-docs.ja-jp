# <a name="renaming-with-a-name-that-exists"></a>存在する名前を使用した名前変更

**ANSI 4.9.4.2** **rename** 関数の呼び出し前に新しい名前のファイルが存在する場合の効果

既存の名前を使用してファイルの名前を変更しようとすると、**rename** 関数は失敗し、エラー コードを返します。
