# <a name="ltdequegt-functions"></a>&lt;deque&gt; 関数

## <a name="swap"></a> スワップ

2 つの deque の要素を交換します。

```cpp
void swap(
    deque<Type, Allocator>& left,
    deque<Type, Allocator>& right,);
```

### <a name="parameters"></a>パラメーター

*左*\
`deque` 型のオブジェクト。

*そうです*\
`deque` 型のオブジェクト。

[deque::swap](../standard-library/deque-class.md#swap) の例を参照してください。
