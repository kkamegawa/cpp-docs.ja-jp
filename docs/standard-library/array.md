# <a name="ltarraygt"></a>&lt;array&gt;

コンテナークラステンプレート**配列**といくつかのサポートテンプレートを定義します。

## <a name="requirements"></a>［要件］

**Header:** \<array>

**名前空間:** std

> [!NOTE]
> @No__t_0array > ライブラリでは、`#include <initializer_list>` ステートメントも使用します。
