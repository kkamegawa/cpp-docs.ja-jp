# <a name="charsformat-class"></a>chars_format クラス

プリミティブの数値変換の浮動小数点形式。

## <a name="syntax"></a>構文

```cpp
enum class chars_format {
    scientific = unspecified ,
    fixed = unspecified ,
    hex = unspecified ,
    general = fixed | scientific
};
```
