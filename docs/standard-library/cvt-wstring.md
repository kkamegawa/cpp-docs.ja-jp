# <a name="ltcvtwstringgt"></a>&lt;cvt/wstring&gt;

以前のバージョンの Visual Studio のヘッダー `<cvt/wstring>` は、stdext:: cvt 名前空間のクラステンプレート[Wstring_convert クラス](../standard-library/wstring-convert-class.md)を定義しています。 ヘッダーは下位互換性のために残されています。 新しいコードでは、`std` 名前空間の [\<locale>](../standard-library/locale.md) で定義されているクラスのバージョンを使用する必要があります。

## <a name="syntax"></a>構文

```cpp
#include <cvt/wstring>
```
