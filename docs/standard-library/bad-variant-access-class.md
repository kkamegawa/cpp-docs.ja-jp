# <a name="badvariantaccess-class"></a>bad_variant_access クラス

バリアント オブジェクトの値に無効なアクセスをレポートには、型 bad_variant_access のオブジェクトがスローされます。

## <a name="syntax"></a>構文

```cpp
class bad_variant_access;
```
