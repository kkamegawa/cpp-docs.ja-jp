# <a name="__int2c"></a>__int2c

**Microsoft 固有の仕様**

割り込みを`2c`トリガーする命令を`int 2c`生成します。

## <a name="syntax"></a>構文

```C
void __int2c(void);
```

## <a name="requirements"></a>必要条件

|組み込み|アーキテクチャ|
|---------------|------------------|
|`__int2c`|x86、x64|
