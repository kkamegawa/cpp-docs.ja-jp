# <a name="odbc-driver-list"></a>ODBC ドライバーの一覧

Visual C では、次のデータベースの ODBC ドライバーを提供します。

- SQL Server

- Microsoft Access

- Microsoft Excel

- dBASE

- Paradox

- Microsoft Oracle ODBC

- テキスト ファイル

ODBC ドライバーから Microsoft および他の企業が、ODBC ドライバーのパックを含む使用可能な情報は、マイクロソフト製品サポート サービスにお問い合わせください。
