# <a name="_unlock"></a>_unlock

マルチスレッドのロックを解放します。

> [!IMPORTANT]
>  これは古い関数です。 Visual Studio 2015 以降、CRT で使用できません。

## <a name="syntax"></a>構文

```
void __cdecl _unlock(
   int locknum
);
```

#### <a name="parameters"></a>パラメーター

*locknum*<br/>
[in] 解放するロックの識別子。

## <a name="requirements"></a>必要条件

**ソース:** mlock.c
