# <a name="null-crt"></a>NULL (CRT)

**NULL** は、多くのポインター操作と関数で使用される Null ポインター値です。 これは 0 と同じです。 **NULL** は、次のヘッダー ファイルで定義されます。CRTDBG.H、LOCALE.H、STDDEF.H、STDIO.H、STDLIB.H、STRING.H、TCHAR.H、TIME.H、WCHAR.H。
