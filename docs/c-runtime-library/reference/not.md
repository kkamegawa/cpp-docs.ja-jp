# <a name="not"></a>not

! 演算子の 。

## <a name="syntax"></a>構文

```C

#define not !
```

## <a name="remarks"></a>Remarks

マクロにより ! 演算子が生成されます。

## <a name="example"></a>例

```cpp
// iso646_not.cpp
// compile with: /EHsc
#include <iostream>
#include <iso646.h>

int main( )
{
   using namespace std;
   int a = 0;

   if (!a)
      cout << "a is zero" << endl;

   if (not(a))
      cout << "a is zero" << endl;
}
```

```Output
a is zero
a is zero
```
