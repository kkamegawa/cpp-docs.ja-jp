# <a name="bitand"></a>bitand

& 演算子の代替手段。

## <a name="syntax"></a>構文

```C

#define bitand &
```

## <a name="remarks"></a>Remarks

マクロにより演算子が生成されます

## <a name="example"></a>例

```cpp
// iso646_bitand.cpp
// compile with: /EHsc
#include <iostream>
#include <iso646.h>

int main( )
{
   using namespace std;
   int a = 1, b = 2, result;

   result = a & b;
   cout << result << endl;

   result= a bitand b;
   cout << result << endl;
}
```

```Output
0
0
```

## <a name="requirements"></a>必要条件
