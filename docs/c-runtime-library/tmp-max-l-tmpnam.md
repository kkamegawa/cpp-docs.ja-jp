# <a name="tmpmax-ltmpnam"></a>TMP_MAX、L_tmpnam

## <a name="syntax"></a>構文

```
#include <stdio.h>
```

## <a name="remarks"></a>解説

`TMP_MAX` は、`tmpnam` 関数で生成できる一意のファイル名の最大数です。 `L_tmpnam` は、`tmpnam` で生成される一時ファイル名の長さです。
