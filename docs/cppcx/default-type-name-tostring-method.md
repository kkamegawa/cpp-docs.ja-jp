# <a name="defaulttypenametostring-method"></a>default::(type_name)::ToString メソッド

現在の型を表す文字列を返します。

## <a name="syntax"></a>構文

```cpp
String^ ToString();
```

### <a name="return-value"></a>戻り値

現在のオブジェクトを表す文字列。

### <a name="requirements"></a>必要条件

**最小値には、クライアントがサポートされています。** Windows 8

**最小値には、サーバーがサポートされています。** Windows Server 2012

**名前空間:** 既定

**ヘッダー:** vccorlib.h
