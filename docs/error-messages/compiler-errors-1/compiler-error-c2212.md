# <a name="compiler-error-c2212"></a>コンパイラ エラー C2212

'identifier': _ _based 指定は関数へのポインターの使用

関数へのポインターを宣言することはできません`__based`します。 コード ベースのデータが必要な場合は、使用、`__declspec`キーワードまたは`data_seg`プラグマ。
