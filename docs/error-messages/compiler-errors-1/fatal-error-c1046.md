# <a name="fatal-error-c1046"></a>致命的なエラー C1046

コンパイラの制限: 構造体の入れ子レベルが深すぎます

構造体、共用体、またはクラスは、15 レベルの入れ子の制限を超えました。 入れ子のレベルを下げるために、定義を書き直してください。 構造体、共用体、またはクラスに 2 つ以上の部分を使用して分割`typedef`を 1 つまたは複数の入れ子になった構造体を定義します。
