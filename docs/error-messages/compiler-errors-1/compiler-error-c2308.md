# <a name="compiler-error-c2308"></a>コンパイラ エラー C2308

一致しない文字列を連結します。

連結の両方の幅と非ワイド文字の文字列が指定されました。 ワイド文字の文字列と非ワイド文字の文字列を連結することはできません。
