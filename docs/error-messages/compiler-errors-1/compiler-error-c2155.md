# <a name="compiler-error-c2155"></a>コンパイラ エラー C2155

'?': 無効な左オペランドの算術演算またはポインター型

式の左側にある`?`0 と比較することはできません。 0 に比較できる演算またはポインター式を使用する必要があります。
