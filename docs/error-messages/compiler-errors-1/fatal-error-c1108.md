# <a name="fatal-error-c1108"></a>致命的なエラー C1108

> DLL が見つかりません: '*dll 名*'

指定された DLL (*dll 名*) パスに見つかりませんでした。

このエラーを解決するには、Visual C を再インストールするか、インストールに適切な .dll ファイルをコンピューターにコピーします。
