# <a name="compiler-error-c2018"></a>コンパイラ エラー C2018

文字 'hexnumber' は認識できません

ソース ファイルには、16 進の番号で識別される、予期しない ASCII 文字が含まれています。 エラーを解決するには、文字を削除します。
