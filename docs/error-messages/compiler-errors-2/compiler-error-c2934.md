# <a name="compiler-error-c2934"></a>コンパイラ エラー C2934

'class': type-class-id が 'identifier' の入れ子になった 'item' として再定義されています

ジェネリック クラスまたはテンプレート クラスを入れ子になった項目として使用することはできません。
