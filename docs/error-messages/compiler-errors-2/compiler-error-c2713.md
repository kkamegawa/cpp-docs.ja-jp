# <a name="compiler-error-c2713"></a>コンパイラ エラー C2713

1 つだけの形式の関数ごとに許可する例外処理

構造化例外処理を使用することはできません (`__try`/`__except`) と C++ 例外処理 (`try`/`catch`) は同じ関数にします。
