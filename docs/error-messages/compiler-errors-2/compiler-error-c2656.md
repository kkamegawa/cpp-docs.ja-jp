# <a name="compiler-error-c2656"></a>コンパイラ エラー C2656

'function': ビット フィールドとして許可されていない関数

関数は、ビット フィールドのメンバーとして宣言されます。

このエラーは、コンス トラクターの初期化子リストで構文エラーが発生することができます。
