# <a name="compiler-warning-level-1-c4403"></a>コンパイラの警告 (レベル 1) C4403

PTR 演算子は正しくありません。

PTR 演算子は、インライン アセンブラー コードで不適切に使用されます。

PTR 演算子は無視されます。
