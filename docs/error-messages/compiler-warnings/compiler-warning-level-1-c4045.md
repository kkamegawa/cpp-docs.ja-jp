# <a name="compiler-warning-level-1-c4045"></a>コンパイラの警告 (レベル 1) C4045

'array': 指定された配列には、初期化子が多すぎます。

配列の初期化子の数が多すぎます。 余分な初期化子は無視されます

配列要素および初期化子のサイズと数が一致することを確認してください。
