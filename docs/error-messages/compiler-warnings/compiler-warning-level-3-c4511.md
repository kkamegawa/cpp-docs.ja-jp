# <a name="compiler-warning-level-3-c4511"></a>コンパイラの警告 (レベル 3) C4511

'class': コピー コンス トラクターを生成できませんでした

コンパイラはクラスの既定のコピー-コンスを生成できませんでした。基底クラスのプライベート コピー コンス トラクターがあります。
