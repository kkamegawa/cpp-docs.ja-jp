# <a name="compiler-warning-level-1-c4031"></a>コンパイラの警告 (レベル 1) C4031

最初のリストよりも長い 2 番目の仮パラメーター リスト

関数は、異なる仮パラメーターで再宣言されています。 コンパイラは、最初の宣言で指定された仮パラメーターを使用します。
