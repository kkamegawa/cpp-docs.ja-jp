# <a name="expression-evaluator-error-cxx0032"></a>式エバリュエーター エラー CXX0032

0 による除算します。

式には、これは無効には、0 による除算が含まれています。 この除数にはリテラルの数字の 0、可能性があります。 または 0 に評価される式がある可能性があります。

このエラーは、can0032 と同じものと同じです。
