# <a name="nmake-fatal-error-u1083"></a>NMAKE の致命的なエラー U1083

ターゲット マクロ 'target' が展開されません。

指定されたターゲットは、マクロが定義されていないか、null 値を持つですの呼び出しです。 NMAKE が null の対象を処理できません。
