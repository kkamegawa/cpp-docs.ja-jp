# <a name="expression-evaluator-error-cxx0072"></a>式エバリュエーター エラー CXX0072

エラー: 型の情報がないか、不明な

.Pch ファイルがリンクされていない、またはコードがない/Zi でコンパイルされたモジュール内にある型への参照。

このエラーは、can0067 と同じものと同じです。
