# <a name="resource-compiler-fatal-error-rc1101"></a>リソース コンパイラの致命的なエラー RC1101

> 指定されていないリソース バイナリ filename

名前変更 (/fo) オプションの後にファイル名がありません。

/Fo オプションには、次の構文を使用します。

> **RC/fo** *出力ファイル名* <em>infilename</em>**.rc**
