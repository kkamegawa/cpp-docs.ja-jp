# <a name="resource-compiler-error-rc2148"></a>リソース コンパイラ エラー RC2148

サブ言語 ID が大きすぎます

サブ言語 ID 値が範囲外でした。

**LANGUAGE** ステートメントでは次の構文を使用しなければなりません。

**LANGUAGE** *primary_language_ID*,*secondary_language_ID*

有効なサブ言語 Id として定義されている**SUBLANG_** WINNT.h ファイルで使用される定数。
