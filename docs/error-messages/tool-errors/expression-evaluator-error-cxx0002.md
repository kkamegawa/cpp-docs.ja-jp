# <a name="expression-evaluator-error-cxx0002"></a>式エバリュエーター エラー CXX0002

エラーにアクセスするユーザーのメモリ

式は、デバッグ中のプログラムに割り当てられていないメモリを参照しようとします。

このエラーは、can0002 と同じものと同じです。
