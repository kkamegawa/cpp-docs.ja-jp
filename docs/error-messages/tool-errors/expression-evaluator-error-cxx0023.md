# <a name="expression-evaluator-error-cxx0023"></a>式エバリュエーター エラー CXX0023

無効な基数です。

C の式エバリュエーターでは、指定された基数は認識されません。 のみ使用できる基数は 10 進数、16 進数、および 8 進数が無効です。

このエラーは、can0023 と同じものと同じです。
