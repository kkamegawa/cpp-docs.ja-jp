# <a name="linker-tools-error-lnk1143"></a>リンカ ツール エラー LNK1143

ファイルが無効または壊れています: COMDAT セクション番号シンボルがありません。

このエラーは、オブジェクト ファイルが壊れている場合に発生することができます。

ファイルを再構築します。
