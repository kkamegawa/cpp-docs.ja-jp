# <a name="nmake-fatal-error-u1001"></a>NMAKE の致命的なエラー U1001

構文エラー: 無効な文字 'character' でマクロ

特定の文字は、マクロが表示されますが、文字、数字、またはアンダー スコアではありません。

このエラーは、マクロの展開で不足している、コロンで発生することができます。

```
syntax error : illegal character '=' in macro
```
