# <a name="resource-compiler-fatal-error-rw1004"></a>リソース コンパイラの致命的なエラー RW1004

予期しないファイルの終わりです

このエラーは、テキストファイルの最終行にラインフィードと復帰文字がないことが原因で発生する場合があります。
