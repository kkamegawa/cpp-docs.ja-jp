# <a name="project-build-error-prj0005"></a>プロジェクト ビルド エラー PRJ0005

ディレクトリ 'directory' に一時ファイルを作成できません。

一時ファイルを作成する呼び出しが失敗しました。 失敗の原因としては次のとおりです。

- 一時ファイル名が不足しました。

- 一時ディレクトリとは、読み取り専用です。

- ディレクトリの temp や TMP 環境変数はありません。

- コンピューターは、空きディスク領域不足がいます。
