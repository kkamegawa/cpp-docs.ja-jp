# <a name="expression-evaluator-error-cxx0048"></a>式エバリュエーター エラー CXX0048

呼び出し順序がサポートされていません

式で指定された関数は、C の式エバリュエーターでサポートされていない呼び出し元のシーケンスを使用します。 ウォッチ ウィンドウで、この関数を呼び出すことはできません。

このエラーは、can0048 と同じものと同じです。
