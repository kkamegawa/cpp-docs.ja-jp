# <a name="resource-compiler-warning-rc4002"></a>リソース コンパイラの警告 RC4002

マクロ 'identifier' に指定された実引数の数が多すぎます

指定された識別子で指定された実際のパラメーターの数が、識別子のマクロ定義内で指定された仮パラメーターの数を超えています。

実際の追加のパラメーターは、収集されたが、マクロの展開時に無視されます。
