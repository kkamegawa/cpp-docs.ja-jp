# <a name="expression-evaluator-error-cxx0006"></a>式エバリュエーター エラー CXX0006

左かっこの欠落

式では、対になっていないかっこが見つかりました。 バランスの取れたかっこを使用して式を再入力します。

このエラーは、can0006 と同じものと同じです。
