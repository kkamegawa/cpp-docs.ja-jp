# <a name="expression-evaluator-error-cxx0012"></a>式エバリュエーター エラー CXX0012

左中かっこがありません。

式には、負荷が不均衡の中かっこが含まれています。 バランスの取れたかっこを使用して式を再入力します。

このエラーは、can0012 と同じものと同じです。
