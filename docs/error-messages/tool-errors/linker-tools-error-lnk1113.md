# <a name="linker-tools-error-lnk1113"></a>リンカ ツール エラー LNK1113

無効なマシン型の型

オブジェクトのヘッダーで指定されたコンピューターの種類が無効です。

このエラーは、ファイルが壊れている場合に発生することができます。 オブジェクトを再構築します。
